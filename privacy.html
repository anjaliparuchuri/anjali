Privacy Policy

This Privacy Policy describes how testsheet collects, uses, and protects user information when using the YouTube Data API v3.

Information We Collect

testsheet may access the following information only after explicit user authorization:

  • YouTube account basic information
  • Channel-related data
  • Video and playlist metadata

The application does not collect passwords or sensitive personal data.

Use of Information

Collected data is used solely to provide the intended functionality of the application, including:

  • Displaying YouTube channel information
  • Performing authorized YouTube API operations
  • Improving app reliability and performance

Data Storage

OAuth access tokens and refresh tokens are stored securely and are used only for authenticated API requests. Data is never shared with third parties.

Data Sharing

testsheet does not sell, rent, or distribute user data to any third party. User data is accessed strictly in accordance with Google API Services User Data Policy.

User Control

Users can revoke access to testsheet at any time through their Google Account security settings. Upon revocation, the application will no longer have access to YouTube data.

Compliance

testsheet complies with the YouTube API Services Terms of Service and Google API Services User Data Policy.

Changes to This Policy

This Privacy Policy may be updated periodically. Any changes will be reflected on this page.
